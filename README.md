📌 Online Aptitude Test System

A desktop-based aptitude testing application built using Java Swing that simulates a real-time exam environment with timer, question navigation, and automated scoring.

🚀 Features:
🎯 Random question selection from multiple subjects (Verbal, Quant, Logical)
⏱️ Countdown timer (30 minutes)
📊 Automatic score calculation
🧭 Question palette for easy navigation
✅ Answer tracking (visited, answered, not answered)
🔄 Section-wise switching (Verbal / Quant / Logical)
🖥️ User-friendly UI using Java Swing

🛠️ Tech Stack
Language: Java
UI Framework: Java Swing
Concepts: OOP, Event Handling, Collections (ArrayList, Shuffle)
Data Storage: Text files (.txt)

📂 Project Structure
OnlineAptitudeSystem.java
verbal.txt
quant.txt
logic.txt

▶️ How to Run
Clone the repository:
git clone https://github.com/your-username/aptitude-system.git
Open in VS Code or any IDE
Compile:
javac OnlineAptitudeSystem.java
Run:
java OnlineAptitudeSystem

📄 Question Format
Each question in .txt files should follow this format:

Question|OptionA|OptionB|OptionC|OptionD|CorrectAnswer
Example:
2+2=?|3|4|5|6|4

This project was developed to simulate a real-world online aptitude test system and improve skills in Java GUI development, logic building, and application design.
